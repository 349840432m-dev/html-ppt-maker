# 版式谱系

| 版式 | 用途 | 使用页码 | 视觉主张 | 动效策略 | PPT 降级 |
|---|---|---|---|---|---|
| hero | 封面/关键主张 |  | 大标题 + 主视觉 | 标题分层入场 | 静态主视觉 + fade |
| section-impact | 章节切换 |  | 视觉隐喻或强对比 | 章节推进 | push/fade |
| framework | 方法论 |  | 三角/飞轮/漏斗/矩阵 | 节点分步出现 | 静态框架图 |
| contrast | 前后对照 |  | 错误与正确并置 | 先错后改 | 拆页或 fade |
| case-walkthrough | 案例拆解 |  | 路径/局部放大 | 按步骤推进 | 拆页 |
| workshop | 练习页 |  | 任务卡/填空框 | 逐项揭示 | 静态任务卡 |
| data-card | 指标/量表 |  | 数字层级 | 数字强调 | 静态指标卡 |
| process-map | 流程 |  | 路线/阶段/闭环 | 路径绘制 | 静态流程图 |
